Lightweight booking plugin for appointments and events with QR code e-tickets.

== Description ==
Kura-ai Booking System (Free) lets you manage appointments and events, accept customer bookings, and generate QR code e-tickets. Includes a setup wizard, admin dashboard, customer dashboard, and email notifications.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/kura-ai-booking-free` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Follow the setup wizard to configure your booking system.

== Frequently Asked Questions ==

= Does this plugin support multiple attendees per booking? =
No, the free version supports single attendee tickets only.

= How are QR code tickets generated? =
Each booking generates a unique ticket ID and QR code PNG.
